# Introducing High School Students to Version Control and CI

## Technologies

- JavaScript
- Node.js
- Jest
- ESLint
- GitHub Actions

## Features

- Addition function
- Multiplication function

## Quality Assurance

- Unit tests with Jest
- Code style verification with ESLint

## Continuous Integration

The GitHub Actions pipeline automatically:

- Installs dependencies
- Runs unit tests
- Runs ESLint
